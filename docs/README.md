# נהיגת חורף בישראל — מדריך אינטראקטיבי לתלמידים

מערכת שיעור דיגיטלית בשלושה חלקים, עם מבחן בסוף כל חלק (4 שאלות רב־ברירה, 4 תשובות בכל שאלה). מיועדת לשליחה לכ־1000 תלמידים.

## קישור ציבורי (GitHub Pages)

**https://dorhorwitznuma.github.io/final_dor/**

האתר פעיל ומתפרסם אוטומטית בכל push ל־main (GitHub Actions).

## מבנה השיעור

| חלק | נושא | מקורות רשמיים |
|-----|------|----------------|
| א׳ | הכנת הרכב לחורף | [משטרת ישראל](https://www.police.gov.il/contentPage.aspx?pid=307), [משרד התחבורה](https://www.gov.il/he/departments/ministry_of_transport_and_road_safety/govil-landing-page) |
| ב׳ | נהיגה בגשם | משטרת ישראל, [רלב"ד Ask](https://ask.ralbad.org.il/309) |
| ג׳ | ערפל, סופות והצפות | משטרת ישראל, רלב"ד — **ללא שלג** |

## התנהגות המבחן

- בסוף כל חלק: **4 שאלות**, בכל שאלה **4 תשובות** (פורמט אמריקאי).
- **תשובה שגויה** → הודעה + חזרה אוטומטית **לתחילת אותו חלק** (יש לקרוא מחדש ולעשות את המבחן שוב).
- **4 תשובות נכונות ברצף** → מעבר לחלק הבא.
- התקדמות נשמרת ב־`localStorage` בדפדפן התלמיד.
